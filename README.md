# 🛡️ Linux OPSEC Templates

Eine Sammlung **praxisgetesteter Linux-Hardening-Configs** als einsatzfertiges Template für OPSEC, Privacy und sichere Server.

## 📂 Inhalt
- 🔐 `sshd_config` – restriktive SSH-Policies
- 🛡️ `sysctl.conf` – Kernel- & Netzwerk-Hardening
- 🔥 `firewall.sh` – Minimalistische iptables-Policy
- 📑 `audit.rules` – Auditd-Logging & Spurenkontrolle
- 📘 `hardening.md` – Schritt-für-Schritt Anleitung

## 🚀 Einsatzgebiete
- Sichere VM-Templates
- Private Anonymity Workflows
- Red Teaming & Pentest-Labs
- Minimalistische Server-Härtung
